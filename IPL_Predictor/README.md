How to Use
Clone the Repository: Get started by cloning this repository to your local machine.
Install Dependencies: Run pip install -r requirements.txt to install all necessary dependencies.
Fire Up the Predictor: Launch the Streamlit web application with streamlit run app.py.
Make Your Predictions: Input match details such as teams and venue, and let our predictor do the rest!
Stay Informed: Receive the predicted probabilities for each team winning the match and make informed decisions.
